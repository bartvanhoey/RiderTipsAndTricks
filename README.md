# Rider Tips & Tricks

## My Favourite Shortcut Keys

| Shortcut Key                 | Action                                                                                   |
| ---------------------------- | -----------------------------------------------------------------------------------------|
| `ALT+SHIFT+ARROW UP/DOWN`    | Moves (selected) line/lines up or down
| `CTRL+D`                     | Duplicate Line
| `CTRL+D, O`                  | Move To Folder
